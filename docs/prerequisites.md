# Prerequisites and optional tools

HoloCore works as a standalone application. You do not need to install Obsidian, Graphify, MemPalace, an AI client, or a remote LLM provider to use its core CLI.

## Required

| Requirement | Why it is needed |
|---|---|
| `uv` | Installs and isolates the HoloCore CLI and MCP server |
| A terminal | Installs HoloCore and runs setup commands |
| Permission to create files in the project | Allows HoloCore to create its local folders, database, graph, and integrations |

Windows with PowerShell is the first-class validated environment. Other operating systems may work, but currently receive less testing.

## Optional

| Tool | When you need it |
|---|---|
| Git | Needed when installing from the repository Git URL before PyPI publication |
| Python 3.11 or newer | Needed only for source-checkout development; `uv tool install` manages the normal tool environment |
| Web browser | Opens the Atlas HTML graph generated by HoloCore |
| Obsidian | Optional visual interface for browsing linked Markdown notes in Archive |
| AI client | Optional; HoloCore also works directly from its CLI |
| OpenAI-compatible LLM | Optional higher-quality chat summarization; the local keyless fallback remains available |

## Which graph do I want to see?

HoloCore provides two related views:

### Atlas: the project structure graph

Atlas maps code, files, functions, and relationships. It does not require Obsidian.

```powershell
cd C:\path\to\project
holocore atlas-refresh
```

Open `<project>\.holocore\atlas.html` in a normal web browser. AI clients consume Atlas through HoloCore's MCP tools and JSON data, so they do not need to open the HTML page.

### Archive: the linked knowledge graph

Archive contains verified Markdown notes, rules, and decisions. Obsidian is optional, but provides a convenient graph view for the links between these notes.

To use it:

1. Install Obsidian from its official website.
2. Choose **Open folder as vault**.
3. Select `<project>/Archive`.
4. Open Obsidian's Graph view.

HoloCore and connected AI clients can still read and search Archive when Obsidian is not installed.

## First-run readiness check

```powershell
uv tool install "git+https://github.com/VenomD846/HoloCore.git"
cd C:\path\to\project
holocore setup
holocore doctor
```

After PyPI publication, `uv tool install holocore` will be the shorter equivalent. Setup creates the Archive, runtime, start-here guide, and supported client integrations non-destructively. Use `holocore paths` to inspect the result.
